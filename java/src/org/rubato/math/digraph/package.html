This package provides an abstract representation of a a Digraph, which is a set of n points, and arrows between some of the points. There is an nxn matrix, each row and column represents whether or not an arrow exists from point i to point j. There is an integer indicating the number of arrows between the given points.

Provided are the name of the Digraph, the number of points in the Digraph, the number of arrows in the Digraph, and the matrix of arrows, explained above.
